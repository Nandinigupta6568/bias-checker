# ⚖️ AI Fairness Auditor

AI Fairness Auditor is a web-based tool that detects bias in datasets and machine learning model predictions. It helps ensure fair and responsible AI decision-making before real-world deployment.

---



## 🚀 Features

- Detects bias across sensitive attributes (e.g., gender)
- Uses Disparate Impact Ratio to measure fairness
- Identifies favored and disadvantaged groups
- Interactive dashboard with visualizations

---

## 📊 Usage

1. Upload a CSV dataset
2. The system will:
   - Analyze dataset bias
   - Analyze model prediction bias
3. View:
   - Approval rates
   - Fairness ratio
   - Bias detection result
   - AI-generated explanation
